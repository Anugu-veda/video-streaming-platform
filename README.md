#Video Streaming Platform

A web-based Video Streaming Platform built using Python, Flask, SQLite, HTML, and CSS.

##Features

- User Signup
- User Login
- Session Management
- Logout Functionality
- Video Upload
- Video Streaming
- SQLite Database Integration
- Protected Routes

##Technologies Used

- Python
- Flask
- SQLite
- HTML
- CSS

##Installation

1. Clone the repository

git clone <repository-url>

2. Install dependencies

pip install flask

3. Run the application

python app.py

4. Open in browser

http://127.0.0.1:5000

##Project Structure

- app.py
- templates/
- static/
- uploads/
- videos.db
